**Overview**

GroomIt360 is a conversational platform built for salons and barbershops.
It helps customers easily book grooming services via WhatsApp and receive friendly reminders when their haircut or treatment is completed.
For owners, it provides a personal dashboard to track appointments, manage staff, monitor profit/loss, control stock, and run payroll â€” moving daily salon operations from spreadsheets/keyboard work into a smooth digital system.
